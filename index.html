Download any video from any site.
Watch later without wifi or cell service.
100% free.

The Download Button

Simply navigate to a page where you can play a video.
Then tap the download button.

Installation

Download YouMacro for Android.
Tap the file to install it.

Permissions

Android 8.0 and higher:
Enable your browser (such as Chrome) to install apps under the Install unknown apps system setting.

Android 7.1.1 and lower:
Enable the Unknown sources setting, found in Settings > Security.
